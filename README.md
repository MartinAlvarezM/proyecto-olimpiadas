# Olympic Games Data Analysis

Este proyecto es un dashboard interactivo que analiza el histórico de medallas de los Juegos Olímpicos. Fue desarrollado como parte de mi transición hacia la **ciencia de datos**, combinando diseño estratégico y análisis visual.

### App en Vivo
Puedes ver el dashboard funcionando aquí: 
[Link a tu App en Render](https://olympics-dashboard-martin.onrender.com/) 

### Tecnologías utilizadas
* **Python**: Lenguaje principal.
* **Streamlit**: Para la creación del dashboard interactivo.
* **Pandas**: Para la limpieza y manipulación de datos (manejo de nulos y tipos de datos).
* **Plotly**: Para las gráficas dinámicas.
* **Render**: Para el despliegue (hosting) de la aplicación.

### Estructura del repositorio
* `app.py`: Código principal de la aplicación Streamlit.
* `notebooks/`: Contiene el análisis exploratorio (EDA) donde se procesaron inicialmente los datos.
* `olympics_data.csv`: Dataset con la información de las medallas.
